# Grade

This teels us the grade assignment as per the marks taken by the student.

Like, 
    if mark is greater than or equal to 90 then the grade is 'A',
    if mark is greater than or equal to 80 then the grade is 'B',
    if mark is greater than or equal to 70 then the grade is 'C',
    if mark is greater than or equal to 50 then the grade is 'D',
    and is the mark is less than 50 then the student failed the examination.
